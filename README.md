# AyaBilA

AyaBilA er en moderne frisør- og kosmetolog-website bygget med Next.js, optimeret til flersprogethed og brugervenlighed.

## 🌟 Features

- **Flersproget support** (dansk, engelsk, arabisk)
- Moderne, responsivt design
- Åbningstider med dynamisk fremhævning af dagsaktuel dag
- Kontaktinformationer og sociale medier
- Google Maps integration
- Komponentbaseret struktur for nem vedligeholdelse

## 🛠️ Tech Stack

- **Next.js** (React framework)
- **TypeScript**
- **SCSS Modules** (modulære styles)
- **ESLint** (kodekvalitet)

## 📁 Projektstruktur

```
├── public/                # Offentlige assets (logo, billeder)
├── src/
│   ├── app/               # Next.js pages og layout
│   ├── components/        # Genanvendelige UI-komponenter
│   ├── context/           # React Context (fx sprog)
│   ├── lib/               # Hjælpefunktioner (fx oversættelser)
│   └── styles/            # SCSS-styles (globale og modulære)
├── package.json           # Afhængigheder og scripts
├── tsconfig.json          # TypeScript konfiguration
├── next.config.ts         # Next.js konfiguration
├── eslint.config.mjs      # ESLint konfiguration
```

## 🚀 Kom i gang

1. **Klon repoet:**
    ```bash
    git clone https://github.com/GR0UD/AyaBilA.git
    cd AyaBilA
    ```
2. **Installer afhængigheder:**
    ```bash
    npm install
    ```
3. **Start udviklingsserver:**
    ```bash
    npm run dev
    ```
4. Åbn [http://localhost:3000](http://localhost:3000) i din browser.

## 📦 Scripts

- `npm run dev` – Start udviklingsserver
- `npm run build` – Byg til produktion
- `npm run start` – Start produktion
- `npm run lint` – Kør ESLint
